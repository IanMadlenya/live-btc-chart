# Live BTC chart
Live Bitcoin price chart using Electron and Highcharts

## Motivation

Building something easy and fun, getting started with Electron.

## Installation
    
    git clone https://github.com/stpoa/live-btc-chart.git
    cd live-btc-chart
    npm install

## Running

    npm run start

## Building

#### MacOS
    npm run build
#### Windows
    npm run build-win
